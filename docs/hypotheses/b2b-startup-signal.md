# B2B-to-Startup 시그널 플랫폼 — 가설 문서

> 생성일: 2026-05-18
> 상태: 🔬 검증 전

## 한 줄 요약
원티드의 채용 데이터를 B2B 세일즈 시그널로 변환해서, B2B 기업이 "지금 뭐가 필요한 스타트업"을 찾고 연락할 수 있게 해주는 서비스

## Lean Canvas

### Problem
- 의사결정자 연락처를 구할 수 없다
- 어떤 회사가 지금 우리 제품이 필요한지 모른다 (구매 타이밍)
- 한국 IT 스타트업 타게팅 DB 자체가 없다
- 콜드 아웃리치 응답률 1~3%로 바닥

### Customer Segment
IT 스타트업에 제품/서비스를 팔고 싶은 B2B 기업의 세일즈/마케팅 담당자
(개발도구, 클라우드, HR SaaS, 오피스, 보안, 법률/회계, 복지, 마케팅 도구 등)

### Unique Value Proposition
원티드의 1차 채용 데이터(HR 행동 로그, 지원자 유입량, 전환율)를 복합 시그널로 조합하여 "이 회사가 지금 뭘 살 타이밍인지" 예측 — 한국 IT 스타트업 집중 커버리지, 직접 경쟁자 없음

### Existing Alternatives
- LinkedIn 수동 검색 (한국 사용자 밀도 낮음)
- 콜드 메일/콜드 콜 (응답률 1~3%)
- 오프라인 행사/컨퍼런스 (비용 높고 스케일 안 됨)
- 지인 네트워크/소개 (확장 불가)
- 글로벌 도구 (Apollo, ZoomInfo — 한국 커버리지 약함)

### Revenue Stream
프리미엄 + 크레딧 하이브리드
- Free: 스타트업 리스트 조회, 기본 필터
- Pro (월 10~20만원/석): 월 100크레딧 포함, HR 연락처 열람/시그널 리포트
- 크레딧 초과 시 건당 1,000~2,000원
- Enterprise: 커스텀 견적

### Price Point
월 10~20만원/석 (Apollo.io~LinkedIn Sales Navigator 사이 구간)

## 핵심 가설

### 🎯 Riskiest Assumption
> "최소 10%의 랜딩페이지 방문자(B2B 세일즈/마케팅 담당자)가 무료 체험 가입 버튼을 클릭할 것이다."

### 보조 가설
1. "최소 5%의 무료 가입자가 유료 크레딧을 결제할 것이다."
2. "최소 15%의 랜딩페이지 방문자가 가격표 섹션을 클릭할 것이다."
